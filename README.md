# Homebrew Tap for Terraspin

Official Homebrew tap for Terraspin.

## Installation

```bash
brew tap terraspin/tap
brew install terraspin
```

## Upgrade

```bash
brew update
brew upgrade terraspin
```

## Uninstall

```bash
brew uninstall terraspin
brew untap terraspin/tap
```

## Source

The Terraspin source code and issue tracker are available at:
